# 第一部分 · 设计篇

## 0. 一句话定位

> **"人人 30 秒就能骑得很帅,但只有高手能骑进排行榜。"**

灵感源于《幽灵行者2》摩托关,原创赛博朋克 IP(不使用其名称/角色/美术)。差异化:Roblox 竞速被"汽车收集+拟真驾驶"占据,技巧流高速摩托跑酷是空白赛道。

## 1. 市场依据(摘要)

约 73% 已验证用户未满 18 岁 → 默认体验无惩罚性死亡;约 80% 会话在移动端 → 输入预算极小,移动端为第一公民;18+ 为增长最快群体(+50% YoY,付费高约 40%,偏好射击/RPG/竞速)→ 技巧深度与竞技排行留住高价值用户;平台规律:**惩罚"高地板",重奖"高天花板"**(Blade Ball/Rivals/Brace 全是低门槛高上限);外观为变现主体,竞技不卖数值(Brace 先例)。**战略:入口泛化、深度分层。**

## 2. 设计支柱(所有决策的裁判)

1. **速度是唯一资源,危险是唯一货币。** 所有机制汇入冲刺能量一条经济链。
2. **低地板,高天花板。** 每个系统都有自动挡兜底和手动挡上限;辅助保安全但必然更慢。
3. **一套内容,三种消费。** 三层玩家骑同一批赛道、同一套手感,差别只在目标框架与失败代价。
4. **计时公平神圣不可侵犯。** 敌人机关固定刷、不卖数值、服务端校验、跨平台输入等价、注入器零容忍。(实现后置 M12/P3.2;V1 期间的义务=不做破坏确定性的设计,地基=ADR-33 定步长)

## 3. 玩家金字塔

### 3.1 底层 —— 休闲兜风/社交(约 70%,主要付费盘)
帅+陪伴。30 秒内爽感堆料;无惩罚体验;摩托收集+全件改装+骑手装扮(步行大厅为展示场);展车/拍照/点赞;每日轻钩子。铁律:进度货币可由休闲玩法获得,永不强制进硬内容。(实现=M13,V1 后)

### 3.2 中层 —— 奖牌收集党(留存脊梁)
四级奖牌:铜(完赛)/银/金/作者牌(可要求零重置,语义见 §4.4)。结算页命门:**"距下一枚奖牌差 X 秒"+一键重开(≤1s)**。幽灵系统(PB/好友/奖牌线)。**分段门实时差值(v0.5 新增):** 赛道内 3~5 个可见霓虹拱门,过门瞬间弹 `-0.42`(绿,领先)/`+0.87`(红,落后),0.8s 淡出,门框同色闪——**参照物永远是"你正在追的那个具体幽灵"**(默认自己 PB;无 PB 则追下一奖牌线幽灵;可切好友/榜首),HUD 角落常驻 `追赶:银牌线`。奖牌阶层叙事只出现在结算页(分段差值明细表=丢时间处一目了然)。无参照的首次自由跑:门只闪色不弹数字(不制造焦虑)。核心能力前 2 小时全解锁(保护榜单)。每周变体挑战(V3)。

### 3.3 顶层 —— 速通/排行(1~3%,内容与传播引擎)
留白进阶技术(点放转向/心流管理/极限走线)→ 攻略视频=免费买量。可信排行榜:赛季+名人堂+榜首回放;反作弊最高优先级(实现后置,规格见 P3.2)。硬核模式=身份标识(一击死,专属外观仅实力可得且大厅可见)。纪录刻名/大屏轮播/月赛。UGC 编辑器为后期王牌。

### 3.4 输送管道
底→中:结算页"差 X 秒"+奖牌掉货币。中→顶:金牌瞬间"超越全球 X%"+榜首幽灵预览。顶→底:硬核外观可见、纪录刻名、最佳跑法轮播。

### 3.5 统一经济
一切活动产出同一货币,流向同一外观商店。

### 3.6 分层实现方式:规则集 × UI × 空间,**不拆地图**
规则集分层(同图:自由跑/奖牌计时/榜单+硬核开关);UI 分层(休闲默认不显计时压力,拿铜牌后渐进展示);空间内分层(主路线宽容,捷径/高架为中顶层风险回报)。唯一专层内容:教程赛道(§8)。否决存档:拆三套图=成本×3 且输送管道断裂。

## 4. 操作与核心系统

### 4.1 移动架构:自由移动 + 隐形样条
样条为数据层,六职责:前方定义、相机参考、导航、重生位姿、检查点门、辅助力场。演出段可局部上轨。

### 4.2 键位总表(每个输入只回答一个问题)

| 输入 | 回答的问题 | 说明 |
|---|---|---|
| 自动前进(赛道) | —— | 无油门,速度节奏权在设计师 |
| **A / D** | 我在哪条线上 | **二元转向,全平台统一,不做摇杆**(输入等价=跨平台榜单公平);0.12s 渐变;点放 feathering=硬核伪模拟技巧;贴墙段内=墙上高度带(§4.9,同一语义的竖直版) |
| **W / Shift** | 花不花速度资源 | 冲刺,**全平台按住**(两键等价;W=平台肌肉记忆兜底:按 W=真的更快=直觉即教学;移动端同为按住,ADR-34)。**空中按住=持续耗能并维持冲刺速;松开=速度衰回巡航且分文不耗——"空中烧不烧能"是真实资源抉择,断桥宽度以此差值设计(ADR-30)**;落地按住无缝续冲;满槽空中点燃同样有效 |

| **攻击键**(点按) | 时机准不准 | **斩击**(出剑):斩敌/劈回子弹/破墙/碎能量核,单键自动锁定;空挥硬直 0.25~0.35s;**任何攻击永不减速**;输入缓冲 0.1s |

**空中主动姿态输入已废(v0.6/ADR-24):** 空中轨迹=起飞质量的结算(ADR-5 推论),技巧重心压在起飞前的速度管理与走线,"起跳后不可挽回"正是起飞决策的重量;完美落地(V2)独立存活;若 V3 跳跃赛道原型滞空太死,再议"俯冲单键"最小复活案。

### 4.3 移动端布局
左拇指=◀▶两个大转向键(按住);右拇指=冲刺(**按住**,与 PC 完全一致,ADR-34)+斩击大键(点按)+轻刹小键(按住)。(右侧竖滑已随空中输入废除,ADR-24)

**⚠️ 待拍板(M10 落移动端前必须,人类裁决):** 按住冲刺与点按斩击同在右拇指——斩击瞬间势必松开冲刺,而 ADR-31 定"松键即熄心流"。候选解:①斩击键移左拇指("斩击点只在直线段"的 §7 lint 恰好保证彼时左手无转向任务);②冲刺松开 ≤0.15s 宽限不熄;③接受"心流中斩击=高操作"。V1 demo 期间 PC 无此冲突,不阻塞。

### 4.4 碰撞与快速重生(v0.6 定稿,ADR-27/28)

**物性声明制(铁律):可骑面是声明的,不是涌现的。** Terrain+带 `Rideable` Tag 的面(路面/坡道/可贴墙面/演出段,TrackBuilder 自动打)=地面白名单,探针只认白名单;**其余一切 BasePart=障碍**,一律走碰撞三档。可骑坡 ≤50°/台阶 1.5 只作用于可骑面内部,不再给任意几何分类(否决存档:涌现分类下同一块石头,矮缓面被当地形→骑上→弹道脱离"弹飞",陡面被当墙→减速,同物不同果=榜单毒药)。**碰撞永不产生垂直速度**——起飞唯一来源=可骑面上的弹道脱离(ADR-5);"弹飞"作为结果类别不存在。

**碰撞三档(判定规格):** 命中瞬间按入射角(+相对速度)一次性定档,不逐帧重判。擦碰(入射角小)=火花+轻减速+镜头震,**纯惩罚不回能**(接触与骑行均不回能,墙=经济中性,ADR-23/37);滑墙=速度投影墙切线、位置永远贴墙外侧,永不向墙内推。硬撞/坠落=0.3~0.5s 故障艺术转场,**转场期间物理冻结积分**。防穿=中心+左右前角三线扫掠取最早命中(侧移也扫)。硬核模式:硬撞=死,回分段门。

**作者牌"零重置"语义(v0.7 定稿):** 零重置=全程**无 R、无锚点重生、无硬撞/坠落转场**;擦碰、滑墙、吃弹不计。计数实现:RaceTimer.resetCount 现只记 R,重生/转场计数钩子归 M4.1。

**快速重生=烘焙锚点链:** 烘焙期(TrackBuilder/Edit 脚本)沿样条每 `Respawn_AnchorSpacingSec`(默认 5,AB 5↔15)秒车程采样,三道 lint(向下射线命中 Rideable/直线段/前方 N studs 无危险物)通过才成为锚点(不可见 Attachment、带弧长、Studio 可手调,10~20 个/条);跳跃段上空无锚点=坠落自动回起跳坡前。运行时:普通模式=回"弧长 ≤ 当前"的最近锚点(切线朝前、贴地、恢复约 60% 速度、0.5s 输入保护,无死亡界面);硬核=回分段门。按弧长回取,不做空间搜索(上下层重叠赛道免疫);**重生弧长永不前进**。**分段门(3~5,可见,计时校验+差值)≠ 重生锚点(不可见,只管重生)**,两套点两个名字,严禁语义纠缠。语义分工:休闲=锚点续骑 / 奖牌党=R 一键重开 / 硬核=分段门重考 / 分段练习(V3)=复用分段门。

### 4.5 冲刺能量经济(v0.5 定稿)

**收入两轨(v0.9 修订,ADR-37):**

| 轨道 | 渠道 | 说明 |
|---|---|---|
| **蓝区·保底(0~80)** | **能量水晶 +18**(CrystalCap=80;**磁吸半径 7=「物理上不可错过」的实现**,开关/半径 AB) | 贴主行车线,顺路骑过即得=向前骑就有口粮(站着一分没有);充能环已删(ADR-37),其保底职能由磁吸承担;新手赛道加密水晶 |
| **红区·技巧(80~100)** | **战斗专营:能量核 +25 / 弹反 +25 / 斩杀 +30 / 闸门 +15**(核/弹反/斩杀破 Cap) | 能量核=悬浮发光多面体(带"需击破"外壳视觉),**斩击判定、无碰撞体**——骑穿未斩=纯机会成本零惩罚,摆偏线与危险位;满槽最后一口必须打出来——**心流=战斗掌握的勋章**(非战斗红区收入已废,ADR-37;完美落地(二期)与近失回能(V2 候选)为未来复活口) |

- **贴墙/墙段=纯移动与路线机制,经济中性(§4.9):** 墙态与任何墙体接触一律零入账;"贴墙就是平地跑"。

- **能量条分段 UI(义务):** 0~80 蓝区(谁都能充)/ 80~100 红区(仅战斗收入,ADR-37)。水晶受 **CrystalCap=80** 约束(理由:守住"心流=胆量勋章",防"捡垃圾进心流"与水晶路线 meta 污染榜单);蓝区充满后再吃水晶给"蓝区已满"轻提示音——**系统可以拒绝,不能沉默**。
- **冲刺规则:** 开局自带 50;耗 22/s;**最小点火时长 0.4s**(任何一次激活至少燃烧 0.4s,不足烧完为止——防碎点冲刺毁视听,封 60Hz 连点邪道);能量=0 按键给**空箱反馈**(干点火咔哒+能量条闪红)。**启动阈值废除**(按了没反应=违反低地板铁律)。
- **心流(ADR-31 定稿):** 满槽 100=点燃资格,不冲刺则**无限期保留**;点燃=按冲刺(空中亦可,ADR-30),数值效果**仅冲刺升档** ×`FlowExtraMultiplier=1.08`(总速 145.8);转向角速度插值分母锚定 135 封顶——**心流不降转向角速度**,唯一手感代价=弯弧 +8%;涌现:心流跳比冲刺跳远 ~17 studs(跳距三档语法:巡航 < 冲刺跳 < 心流跳)。**结束条件仅三类:燃尽**(`FlowExitEnergy=0` 烧完整管,AB 60=状态式)**/ 松键或刹车**(冲刺暂停即熄,再进需重新满槽——"点燃时机"就是决策)**/ 硬撞坠落;滞空、吃弹、擦墙均不断**(吃弹已由减速惩罚,ADR-31)。最后 20 点必须靠战斗收入(CrystalCap=80,ADR-37)。表现:色调+音乐变奏+近肩相机+满槽 READY 脉冲。
- **禁令与阀门:** 静止/无行为回能**永久禁止**;`Energy_MoveRegenPerSec`(骑行涓流)为**默认关闭的 AB 阀**,仅当埋点"底层玩家能量归零时长占比"证明饥荒时启用。

### 4.6 攻击系统:斩击,一刀多结算(v0.5 定稿)

**正名:** 主动词是**出剑**,"格挡"语义废弃——你攻击子弹(把它劈回去),而不是防御子弹。同一刀在不同语境结算不同结果,叙事与手感全程是进攻。

**判定实现规格(铁律):** 所有可斩目标用 CollectionService Tag 注册为脚本对象;按键帧对注册目标做**车体局部空间体积查询**(前 12 / 左右 ±7 / 上下 ±5 / 后 -2,纯数学距离判断)——**禁用物理碰撞检测**(锚定件 Touched 不触发,且物理接触受帧率影响破坏确定性)。剑光=纯 VFX,与判定零耦合;子弹=固定弹道脚本对象,弹反=改写其目标为射手。白送性质:窗口时长=体积深度÷车速,越快窗口越紧,技巧要求随速度自动升级。单次斩击结算体积内**全部**有效目标(多杀=关卡师排布的高光)。服务端按目标位置+复现位置做距离合理性校验(M12)。

**反馈铁律(v0.6/ADR-20):反馈永不冻结画面与输入。** gameplay 内禁止子弹时间/hitstop/任何时间膨胀——135 速下 0.1s 冻结=偷走 ~13 studs 读路窗口,且打断转向渐变与下一斩击窗口的预判;局部时间膨胀污染计时神圣与幽灵/共存同步。弹反爽感=非时间性反馈:音效尖峰+闪光/色差脉冲+子弹肉眼可见飞回杀射手的演出(能量涌动在击杀帧,ADR-29)。慢镜头只允许出现在结算页与回放。与"任何攻击永不减速"同源。

**快照弹道(v0.6/ADR-21):** 子弹=发射瞬间固定弹道快照,永不追踪——变速(冲刺/轻刹)与变线本身就是闪避,"加速=闪避"与支柱 1 自洽。

**敌人物性(v0.6/ADR-22):** 远程敌本体=无碰撞脚本对象,骑穿零惩罚,威胁全部来自子弹;可摆路面正中(迎面弹=最可读的"物体飞向我"弹道,强化弹反)。**生死线:敌人身体永不做碰撞障碍**——否则即 ADR-18 障碍型近战敌借尸还魂。

| 目标 | 窗口 | 结算 | 能量 |

### 4.9 贴墙段(v0.8 定稿,ADR-36;实现=M6.5)
**形态=授权段式表面吸附(GR2 摩托/MK8 反重力语法):** 可贴墙面=带 `WallRideSurface` Tag 的授权墙段(直墙先行,曲墙随 TrackBuilder P3 喂样条侧偏框架;倒挂/天花板/管道仍属 V3 上轨演出段,ADR-26)。**磁吸模型:** 墙态内 down=−墙法线,速度不变;A/D=墙上高度带;进入=坡唇/触发区(地面或空中皆可,0.2s 位置姿态混合,有速度阈值);退出=段尾自动弹道交接 / 驶下下缘回地 / 驶出上缘弹道——**墙上高度=落差跳距**(§7 跳跃语法联动,墙段尾高位弹出=高台弹射)。相机 roll≤90° 平滑滚转,FOV 不变。
**经济与判定(v0.9,ADR-37):** 墙=**经济中性**——墙态与任何墙体接触(未进入墙态)一律零入账,"贴墙就是平地跑";接触=纯惩罚照旧。进出与墙上滞空**不断心流**(只是移动状态);坠落按 fell 断。墙段**不烘焙重生锚点**(坠落回段前锚点,同跳跃段);硬核=回分段门。
**存在理由(soft lint,§7):** 无收入之后,墙段必须在**路线上有意义**——捷径/更顺的走线/高台弹射/躲弹/演出,否则无人上墙(原"风险 lint"随收入删除一并撤销:无收入=无农场);墙段内不摆判定点与射手(不变)。

## 7. 关卡设计语法(TrackBuilder 的 lint 清单)

- 高速段宽走廊;截面微谷形;地标+灯带导航;可贴墙面统一霓虹条。
- 捷径全部是有意设计的风险-回报缺口;好捷径入口配挡板(技巧税)。
- **斩击/弹反判定点只放直线段,永不与转向决策重叠**(同时决策数峰值锁 2;移动端右拇指永不双键)。
- **主路线永不强制斩击(v0.6/ADR-21):** 强制斩击仅允许教程教学闸门(教学锚)与捷径挡板(技巧税);路面敌与能量闸门必须可绕行或可穿透(闸门不劈=擦碰级减速,不挡死)。
- **物性 lint(v0.6/ADR-28):** 一切可骑面必须带 Rideable Tag(TrackBuilder 自动);路面与主线旁装饰默认障碍;禁止"骑不上去又不算墙"的模糊几何。
- 水晶贴主行车线(**磁吸=保底的实现**:主线水晶+磁吸=顺路即得,开关/半径 AB;充能环已删,ADR-37);**能量核摆偏线与危险位**;新手赛道加密水晶、硬核赛道多核——能量供给密度=关卡难度旋钮之一。
- **能量供给两桶 lint(v0.9 修订,ADR-37):** 供给密度统计按 蓝区(水晶×磁吸)/ 红区(战斗:核/敌/闸门) 两桶分列;**红区=0 触发警告**(心流不可达);参考密度:晶 12~18 / 核 3~6 / 敌 3~5 / 闸 0~2(M9.5 标定);蓝区密度=底层冲刺口粮,埋点"能量归零时长占比"为阀门信号(ADR-17)。

- **跳跃段语法(v0.7.1 新增,ADR-5/30/31 的设计尺子):** 同高落点近似跳距 ≈ 2·v²·tanθ/g(g=260,水平速度全额保留):20° 坡≈巡航 28/冲刺 51/心流 60 studs;30° 坡≈44/81/94。**平断口(θ=0)同高对岸永远跳不过**(vy0=0),断桥仅三种合法形态:①平断口+对岸下沉(跳距≈v·√(2h/g))②断口带 5~15° 起跳唇 ③坡顶断口(Rig3 形态)。**缺口设计法:巡航过不去/冲刺刚好/心流余量**;捷径口可用"心流才过"档(空中松开冲刺=距离骤减,ADR-30 的抉择即难度)。精确数字跑 `tools/jumptable.lua`(读 Config 实时算);跳跃段上空无锚点=坠落回起跳坡前(ADR-27)。
