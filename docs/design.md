# 第一部分 · 设计篇

## 0. 一句话定位

> **"人人 30 秒就能骑得很帅,但只有高手能骑进排行榜。"**

灵感源于《幽灵行者2》摩托关,原创赛博朋克 IP(不使用其名称/角色/美术)。差异化:Roblox 竞速被"汽车收集+拟真驾驶"占据,技巧流高速摩托跑酷是空白赛道。

## 1. 市场依据(摘要)

约 73% 已验证用户未满 18 岁 → 默认体验无惩罚性死亡;约 80% 会话在移动端 → 输入预算极小,移动端为第一公民;18+ 为增长最快群体(+50% YoY,付费高约 40%,偏好射击/RPG/竞速)→ 技巧深度与竞技排行留住高价值用户;平台规律:**惩罚"高地板",重奖"高天花板"**(Blade Ball/Rivals/Brace 全是低门槛高上限);外观为变现主体,竞技不卖数值(Brace 先例)。**战略:入口泛化、深度分层。**

## 2. 设计支柱(所有决策的裁判)

1. **速度是唯一资源,危险是唯一货币。** 所有机制汇入冲刺能量一条经济链。
2. **低地板,高天花板。** 每个系统都有自动挡兜底和手动挡上限;辅助保安全但必然更慢。
3. **一套内容,三种消费。** 三层玩家骑同一批赛道、同一套手感,差别只在目标框架与失败代价。
4. **计时公平神圣不可侵犯。** 敌人机关固定刷、不卖数值、服务端校验、跨平台输入等价、注入器零容忍。

## 3. 玩家金字塔

### 3.1 底层 —— 休闲兜风/社交(约 70%,主要付费盘)
帅+陪伴。30 秒内爽感堆料;无惩罚体验;摩托收集+全件改装+骑手装扮(步行大厅为展示场);展车/拍照/点赞;每日轻钩子。铁律:进度货币可由休闲玩法获得,永不强制进硬内容。

### 3.2 中层 —— 奖牌收集党(留存脊梁)
四级奖牌:铜(完赛)/银/金/作者牌(可要求零重置)。结算页命门:**"距下一枚奖牌差 X 秒"+一键重开(≤1s)**。幽灵系统(PB/好友/奖牌线)。**分段门实时差值(v0.5 新增):** 赛道内 3~5 个可见霓虹拱门,过门瞬间弹 `-0.42`(绿,领先)/`+0.87`(红,落后),0.8s 淡出,门框同色闪——**参照物永远是"你正在追的那个具体幽灵"**(默认自己 PB;无 PB 则追下一奖牌线幽灵;可切好友/榜首),HUD 角落常驻 `追赶:银牌线`。奖牌阶层叙事只出现在结算页(分段差值明细表=丢时间处一目了然)。无参照的首次自由跑:门只闪色不弹数字(不制造焦虑)。核心能力前 2 小时全解锁(保护榜单)。每周变体挑战。

### 3.3 顶层 —— 速通/排行(1~3%,内容与传播引擎)
留白进阶技术(点放转向/心流管理/极限走线)→ 攻略视频=免费买量。可信排行榜:赛季+名人堂+榜首回放;反作弊最高优先级。硬核模式=身份标识(一击死,专属外观仅实力可得且大厅可见)。纪录刻名/大屏轮播/月赛。UGC 编辑器为后期王牌。

### 3.4 输送管道
底→中:结算页"差 X 秒"+奖牌掉货币。中→顶:金牌瞬间"超越全球 X%"+榜首幽灵预览。顶→底:硬核外观可见、纪录刻名、最佳跑法轮播。

### 3.5 统一经济
一切活动产出同一货币,流向同一外观商店。

### 3.6 分层实现方式:规则集 × UI × 空间,**不拆地图**
规则集分层(同图:自由跑/奖牌计时/榜单+硬核开关);UI 分层(休闲默认不显计时压力,拿铜牌后渐进展示);空间内分层(主路线宽容,捷径/高架为中顶层风险回报)。唯一专层内容:教程赛道(§8)。否决存档:拆三套图=成本×3 且输送管道断裂。

## 4. 操作与核心系统

### 4.1 移动架构:自由移动 + 隐形样条
样条为数据层,六职责:前方定义、相机参考、导航、重生位姿、检查点门、辅助力场。演出段可局部上轨。

### 4.2 键位总表(每个输入只回答一个问题)

| 输入 | 回答的问题 | 说明 |
|---|---|---|
| 自动前进(赛道) | —— | 无油门,速度节奏权在设计师 |
| **A / D** | 我在哪条线上 | **二元转向,全平台统一,不做摇杆**(输入等价=跨平台榜单公平);0.12s 渐变;点放 feathering=硬核伪模拟技巧 |

### 4.3 移动端布局
左拇指=◀▶两个大转向键(按住);右拇指=冲刺(**点按切换**,PC 按住 Shift,设置可互换)+斩击大键(点按)+轻刹小键(按住);右侧竖滑(二期)=空中姿态。"斩击点只在直线段"保证右拇指永不双键同按。

### 4.4 碰撞三档 + 快速重生
擦碰(入射角小):火花+轻减速+镜头震,继续跑,该区=擦墙回能区。硬撞/掉出:0.3~0.5s 故障艺术转场→样条最近点重生、朝前、恢复约 60% 速度,无死亡界面。硬核模式:硬撞=死,回检查点。作者牌可要求零重置。

### 4.5 冲刺能量经济(v0.5 定稿)

**收入四象限(封版):**

| | 被动获取 | 主动执行 |
|---|---|---|
| **保底(蓝区)** | **充能环**:横跨路面的拱门,物理上不可能错过——向前骑即有保底收入(本质=前进门控的回能,站着一分没有) | **能量水晶** +18:贴主行车线,**磁吸半径 7 studs** 自动飞向玩家;骑过即收,不占斩击键 |
| **技巧(可入红区)** | 擦墙 +8 / 贴墙 +12每秒 /(二期)完美落地 | **能量核 +25**:悬浮发光多面体(比水晶大、带"需击破"外壳视觉),**斩击判定、无碰撞体**——骑穿未斩=纯机会成本零惩罚;摆放偏线与危险位(贴墙段末端/捷径口旁/跳台抛物线顶点);**弹反 +25** |

- **能量条分段 UI(义务):** 0~80 蓝区(谁都能充)/ 80~100 红区(仅技巧收入)。水晶受 **CrystalCap=80** 约束(理由:守住"心流=胆量勋章",防"捡垃圾进心流"与水晶路线 meta 污染榜单);蓝区充满后再吃水晶给"蓝区已满"轻提示音——**系统可以拒绝,不能沉默**。
- **冲刺规则:** 开局自带 50;耗 22/s;**最小点火时长 0.4s**(任何一次激活至少燃烧 0.4s,不足烧完为止——防碎点冲刺毁视听,封 60Hz 连点邪道);能量=0 按键给**空箱反馈**(干点火咔哒+能量条闪红)。**启动阈值废除**(按了没反应=违反低地板铁律)。
- **心流:** 满槽 100 持续冲刺进入(色调+音乐变奏+小幅加成,碰撞即断);最后 20 点必须靠技巧收入。
- **禁令与阀门:** 静止/无行为回能**永久禁止**;`Energy_MoveRegenPerSec`(骑行涓流)为**默认关闭的 AB 阀**,仅当埋点"底层玩家能量归零时长占比"证明饥荒时启用。

### 4.6 攻击系统:斩击,一刀多结算(v0.5 定稿)

**正名:** 主动词是**出剑**,"格挡"语义废弃——你攻击子弹(把它劈回去),而不是防御子弹。同一刀在不同语境结算不同结果,叙事与手感全程是进攻。

**判定实现规格(铁律):** 所有可斩目标用 CollectionService Tag 注册为脚本对象;按键帧对注册目标做**车体局部空间体积查询**(前 12 / 左右 ±7 / 上下 ±5 / 后 -2,纯数学距离判断)——**禁用物理碰撞检测**(锚定件 Touched 不触发,且物理接触受帧率影响破坏确定性)。剑光=纯 VFX,与判定零耦合;子弹=固定弹道脚本对象,弹反=改写其目标为射手。白送性质:窗口时长=体积深度÷车速,越快窗口越紧,技巧要求随速度自动升级。单次斩击结算体积内**全部**有效目标(多杀=关卡师排布的高光)。服务端按目标位置+复现位置做距离合理性校验(M12)。

**交互矩阵:**

| 目标 | 窗口 | 结算 | 能量 |
|---|---|---|---|
| 远程敌·子弹 | 时间(到达瞬间,预告线) | 劈回子弹→击杀射手 | +25(破 Cap) |
| 远程敌·本体 | 空间(骑到近前) | 直接斩杀(凡敌皆可斩,绕后是其惩罚) | +30(破 Cap) |
| 能量核 | 空间 | 击碎收能 | +25(破 Cap) |
| 能量闸门 | 空间 | 劈开通行(不劈=按擦碰减速) | +15 |
| 捷径挡板 | 空间 | 劈开进捷径 | 0(捷径即奖励) |
| 水晶 | —— | 骑过自动收(斩中也算收) | +18(蓝区) |
| 墙 | **不判定** | 剑光扫过冒火花(纯装饰);墙的能量归擦墙系统专营,防"斩墙农场"污染擦碰经济 | 0 |
| 幽灵/共存玩家 | 不判定 | 穿透 | 0 |

子弹命中(未弹反)=减速一档,不死。失误的代价只在"错过能量",永不在车速(空挥硬直除外)。

**废案存档(勿复活):** 枪械(瞄准=第二方向输入,移动端地板塌;抢读路注意力;弹反尖峰爽感优于射击摊平,且"劈回子弹杀射手"已含远程幻想);**障碍型近战敌**(负反馈链:0.3s 硬直@135 速=40 studs 输入锁死区,"提前按→空挥→硬直中撞敌→减速"因果稀烂;且"冲向静止物"是最难读的窗口,远劣于"物体飞向我"的弹反)——其机制位由能量核取代,其"斩敌"幻想由远程敌本体可斩承担;击飞物理与敌人主动攻击玩家永久排除(敌人是赛道元素,不是 AI);按住格挡(占用持续状态、奖励消极提前按、时机技巧蒸发);靠近自动破墙(按键时机就是玩法本体)。**V2 可选:** 判定窗口内"完美"内圈(+50% 能量+专属音画,Blade Ball 式天花板)。

### 4.7 可破坏墙家族(v0.5 排序)
- **捷径挡板=承重墙,不可砍:** 让捷径成为"带技巧税的选择"的那道税;也是节点墙的地基。
- **教学闸门=斩击键的教学锚:** 教程赛道横路一面,保证 100% 玩家第一分钟学会此键。
- **横路收费站式闸门=最弱形态:** "要按键的减速带",正式赛道少用当调味。
- **V2·前置节点联动墙(GR2 电池语法的正统移植):** 斩路边发光节点→电缆光束冲向前方墙→墙开启 N 秒(墙面倒计时灯条)→闭合。买到:编织走线(节点与墙异线)、提前读路、开窗时限=速度压力;与捷径挡板叠加=风险回报的平方。**视觉铁律:节点与墙用发光电缆物理连接+同色编码**,135 速下一眼可读。`Combat_GateLinkWindowSec=2.5`。

### 4.8 转向辅助开关(底层安全网)
自动修线+提前减速;保安全但必然更慢——"关辅助"自然成为底→中的仪式。

## 5. 手感与数值基准(Config 实时可调)

横向:渐入 0.12s/回正 0.20s,转向角速度 110↔55°/s 随速度插值。速度:巡航 100,冲刺 ×1.35,心流 ×1.08;提速 0.25s/降速 0.35s;轻刹下限 70%(渐入 0.15s)。能量:上限 100/开局 50/耗 22每秒/水晶 18(Cap 80,磁吸 7)/核 25/弹反 25/斩杀 30/闸门 15/擦墙 8/贴墙 12每秒/最小点火 0.4s。倾斜:车体 15~25°+骑手 5~10°,回正过冲;落地压悬挂、起跳微抬头。物理:重力 260/可骑坡 ≤50°/台阶 1.5/起飞灵敏度 0.25。战斗:判定盒前 12 侧 7 上下 5 后 2/缓冲 0.1s/硬直 0.3s。声音三层:引擎随速升调/冲刺点火/近距呼啸。

## 6. 相机与视角
默认第三人称+动态档位:常规(FOV72,距 16)→冲刺档(压低拉近 FOV88+速度线微震)→心流档(近肩)。一人称=设置项(PC 优先)。理由存档:平台习惯/**皮肤可见性=付费盘生死线**/移动端晕动症/擦墙间隙可读性。弹簧跟随滞后 0.1s、转向慢半拍、撞墙回退防穿。

## 7. 关卡设计语法(TrackBuilder 的 lint 清单)

- 高速段宽走廊;截面微谷形;地标+灯带导航;可贴墙面统一霓虹条。
- 捷径全部是有意设计的风险-回报缺口;好捷径入口配挡板(技巧税)。
- **斩击/弹反判定点只放直线段,永不与转向决策重叠**(同时决策数峰值锁 2;移动端右拇指永不双键)。
- 水晶贴主行车线(磁吸 7 兜漏接);充能环横跨路面(保底);**能量核摆偏线与危险位**;新手赛道多环、硬核赛道多核——能量供给密度=关卡难度旋钮之一。
- 弯道按"全速+好走线可过"设计,刹车永远只是容错余量不是通关前提。
- 敌人密度:普通赛道 3~5,固定摆放确定性刷新;R 重开全部重置(验收断言)。
- **分段门 3~5 个/条(60~90s 赛道):** 可见霓虹拱门,兼任服务端校验门与岔路导航;差值显示规格见 §3.2。
- (V2)节点-墙电缆连接+同色编码为视觉铁律。
- 待定:爽段/技巧段/演出段配比模板。

## 8. 大厅与进入流程
**原则:进门即玩,永不等人,没有"房间"概念,任何菜单都是流失点。**
步行大厅(Avatar 步行=平台社交母语+骑手装扮展示场;直径 ≤200 studs);门=空间化选关(锁门显示解锁条件);**进门自动上车仪式**:穿门→0.5s 上车动画→滑行至发车线→**3-2-1 固定点火自动起步**(发车标准化:服务端计时自绿灯起算,无抢跑,首个决策=第一个弯的走线);**首登直进教程赛道**(60~90s 剧本:转向→擦墙→冲刺→斩击,保送铜牌;带奖牌与货币初见大厅;设跳过口);老玩家出生大厅面朝大门广场;单人练习=每条赛道默认态(幽灵陪跑,同服半透明共存各自计时);结算页主按钮=一键重开;训练场=一扇普通门(无计时,水晶/可擦墙/弹反木桩);竞技场门(V3)整点回合、等待区可热身;好友加入/VIP 私服走平台原生。分段练习(检查点起跑)记 V3。

## 9. 社交三层
洞察:无碰撞下"实时竞速"≈"同步播放的真人幽灵",三层共用地基。L1 异步幽灵(V1~V2,复用反作弊回放系统;变强引擎)→ L2 无碰撞共存(V2 随 M12;氛围与橱窗,半透明各自计时)→ L3 定时回合赛+月杯赛(V3,赛事包装)。**永久排除带碰撞对抗**。UI 走聪明默认值,不做幽灵管理界面;二选一留幽灵。

## 10. 商业化与公平
