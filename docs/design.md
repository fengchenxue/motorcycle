# 第一部分 · 设计篇

## 0. 一句话定位

> **"人人 30 秒就能骑得很帅,但只有高手能骑进排行榜。"**

灵感源于《幽灵行者2》摩托关,原创赛博朋克 IP。差异化:Roblox 竞速被"汽车收集+拟真驾驶"占据,技巧流高速摩托跑酷是空白赛道。

## 1. 市场依据(摘要)

~73% 用户未满 18 → 默认体验无惩罚性死亡;~80% 会话在移动端 → 输入预算极小,移动端第一公民;18+ 增长最快且付费更高 → 技巧深度与竞技留住高价值用户;平台规律:惩罚高地板、重奖高天花板;外观为变现主体,竞技不卖数值。**战略:入口泛化、深度分层。**

## 2. 设计支柱(所有决策的裁判)

1. **速度是唯一资源,危险是唯一货币。** 所有机制汇入冲刺能量一条经济链。
2. **低地板,高天花板。** 每个系统都有自动挡兜底和手动挡上限;辅助保安全但必然更慢。
3. **一套内容,三种消费。** 三层玩家同赛道同手感,差别只在目标框架与失败代价。
4. **计时公平神圣不可侵犯。** 敌人机关固定刷、不卖数值、服务端校验、跨平台输入等价(实现后置 M12/P3.2;V1 义务=不做破坏确定性的设计,地基=E2 定步长)。

## 3. 玩家金字塔

- **底层·休闲兜风(~70%,主要付费盘):** 帅+陪伴;30 秒爽感;无惩罚体验;收集/改装/装扮(M13,V1 后)。铁律:进度货币可由休闲玩法获得,永不强制进硬内容。
- **中层·奖牌党(留存脊梁):** 四级奖牌=铜(完赛)/银/金/作者牌(可要求零重置,§4.4)。结算页命门:**"距下一枚奖牌差 X 秒"+一键重开(≤1s)**。**分段门实时差值:** 3~5 个可见霓虹拱门,过门弹 `-0.42`(绿,领先)/`+0.87`(红,落后),0.8s 淡出,门框同色闪;**参照=正在追的单一具体幽灵**(默认自己 PB,无则下一奖牌线,可切好友/榜首),HUD 常驻`追赶:银牌线`;无参照首跑只闪色不弹数字;奖牌阶层叙事只进结算页(分段差值明细)。核心能力前 2 小时全解锁(保护榜单)。
- **顶层·速通排行(1~3%,内容与传播引擎):** 点放转向/心流管理/极限走线=留白技术 → 攻略视频=免费买量;可信榜单(赛季/名人堂/榜首回放);硬核模式=身份标识(一击死,专属外观仅实力可得);UGC 编辑器=后期王牌。
- **输送管道:** 底→中=结算"差 X 秒"+奖牌掉货币;中→顶=金牌瞬间"超越全球 X%"+榜首幽灵预览;顶→底=硬核外观可见/纪录刻名/最佳跑法轮播。一切产出同一货币,流向同一外观商店。
- **分层=规则集 × UI × 空间,不拆地图:** 同图三模式(自由跑/奖牌计时/榜单+硬核);UI 渐进显示计时压力;主路线宽容+捷径高架高危。唯一专层内容=教程赛道(§8)。

## 4. 操作与核心系统

### 4.1 移动架构:自由移动+隐形样条
样条=数据层,六职责:前方定义/相机参考/导航/重生位姿/检查点门/辅助力场。演出段可局部上轨(V3)。

### 4.2 键位总表(每个输入只回答一个问题)

| 输入 | 回答的问题 | 说明 |
|---|---|---|
| 自动前进 | —— | 无油门,速度节奏权在设计师 |
| **A / D** | 我在哪条线上 | 二元转向,全平台统一,不做摇杆(输入等价=跨平台公平);0.12s 渐变;点放 feathering=硬核技巧;贴墙段内=墙上高度带(§4.9) |
| **W / Shift** | 花不花速度资源 | 冲刺,**全平台按住**(两键等价,W=肌肉记忆兜底)。**空中按住=持续耗能维持冲刺速;松开=衰回巡航且不耗能——断桥宽度以此差值设计**;落地按住无缝续冲;满槽空中点燃有效 |
| **S** | 用不用时间换容错 | 轻刹至 70%,0.15s 渐入松开速回;与冲刺同按刹车优先且不耗能(实验开关,M9.5 AB 终审) |
| **攻击键**(点按) | 时机准不准 | **斩击**(出剑):斩敌/劈回子弹/破墙/碎核,单键自动锁定;空挥硬直 0.3s;**任何攻击永不减速**;缓冲 0.1s |
| **R** | —— | 任意时刻重开(回发车线,计入重置次数,场上目标全重置) |

空中无主动姿态输入:空中轨迹=起飞质量的结算,技巧压在起飞前的速度管理与走线。

### 4.3 移动端布局(ADR-39 定稿)
默认两拇指:左拇指=◀▶转向(按住);右拇指=冲刺(按住)+斩击大键(点按)+轻刹小键(按住)。
**冲刺松键宽限 `Energy_ReleaseGraceSec=0.12`(AB 0.1↔0.15,全平台一致):** 松键 ≤宽限内重新按住则心流不断(能量与速度仍按真实按键状态走;超时按松键处理)——右拇指点斩击不再误熄心流,PC 手滑同受益,"点燃时机=决策"保留。
**M10 义务:键位自定义 + 官方"爪握预设"**(斩击键置右上角供食指点按,右拇指专注冲刺/刹车——高阶布局,不作默认门槛)。斩击移左拇指已否决(挤压偏线核"调线→斩"的衔接,且不合平台习惯)。

### 4.4 碰撞与快速重生

**物性声明制:可骑面是声明的,不是涌现的。** 可骑面=Terrain+`Rideable` Tag 白名单(TrackBuilder 自动打);**其余一切 BasePart=障碍**,走碰撞三档;可骑坡 ≤50°/台阶 1.5 只作用于白名单内部。**碰撞永不产生垂直速度**——起飞唯一来源=可骑面上的弹道脱离;"弹飞"作为结果类别不存在。

**碰撞三档:** 命中瞬间按入射角一次性定档,不逐帧重判。擦碰(<20°)=火花+轻减速+镜头震,纯惩罚零回能;弹开(≥20°)=反射方向+泄速 25%(低速陡角降级为弹开);硬撞(≥55° 且速度≥20)=0.3~0.5s 故障艺术转场(**转场期间物理冻结积分**)→重生 60% 速。滑墙=速度投影墙切线、位置永远贴墙外侧,永不向墙内推。防穿=中心+左右前角三线扫掠取最早命中。硬核模式:硬撞=死,回分段门。

**作者牌"零重置":** 全程无 R、无锚点重生、无硬撞/坠落转场;擦碰滑墙吃弹不计(重生计数钩子归 M4.1)。

**快速重生=烘焙锚点链:** 烘焙期沿样条每 `Respawn_AnchorSpacingSec`(5,AB 5↔15)秒采样,三道 lint(下射线命中 Rideable/直线段/前方无危险物)→不可见锚点(带弧长,10~20/条);**跳跃段与墙段上空无锚点**=坠落回段前。运行时:普通=回"弧长≤当前"最近锚点(恢复 60% 速,0.5s 输入保护,无死亡界面);硬核=回分段门;按弧长回取不做空间搜索(上下层重叠免疫),**重生弧长永不前进**。**分段门(可见,计时校验)≠重生锚点(不可见,只管重生),两套点两个名字,严禁语义纠缠。** 语义分工:休闲=锚点续骑/奖牌党=R 重开/硬核=分段门重考/分段练习(V3)=复用分段门。

### 4.5 冲刺能量经济(两轨制)

| 轨道 | 渠道 | 说明 |
|---|---|---|
| **蓝区·保底(0~80)** | **水晶 +18**(CrystalCap=80;**磁吸半径 7=「物理上不可错过」的实现**,开关/半径 AB) | 贴主行车线,顺路骑过即得=向前骑就有口粮;新手赛道加密水晶 |
| **红区·技巧(80~100)** | **战斗专营:核 +25 / 弹反 +25 / 斩杀 +30 / 闸门 +15**(前三破 Cap) | 核=悬浮发光多面体(带"需击破"外壳),斩击判定、无碰撞,骑穿=零惩罚纯机会成本,摆偏线与危险位;满槽最后一口必须打出来——**心流=战斗掌握的勋章** |

- **贴墙/墙段与一切墙体接触=经济中性零入账**(§4.9);静止/无行为回能永久禁止;`Energy_MoveRegenPerSec`=默认关的 AB 阀(仅当埋点"能量归零时长占比"证明饥荒才开)。
- **能量条分段 UI:** 0~80 蓝 / 80~100 红(仅战斗收入);蓝区满后再吃水晶给"蓝区已满"轻提示——**系统可以拒绝,不能沉默**。
- **冲刺:** 开局 50;耗 22/s;**最小点火 0.4s**(不足烧完为止,防碎点冲刺与连点邪道);能量=0 按键给空箱反馈(咔哒+条闪红);启动阈值不存在。
- **心流:** 满槽 100=点燃资格(不冲刺无限期保留);点燃=按冲刺(空中亦可);效果=冲刺再 ×1.08(总速 145.8),**转向角速度分母锚 135 不降**,唯一手感代价=弯弧 +8%;心流跳比冲刺跳远 ~17 studs(跳距三档:巡航<冲刺跳<心流跳)。**结束仅三类:燃尽(`FlowExitEnergy=0` 烧完整管,AB 60)/ 松键(>0.12s 宽限,ADR-39)或刹车(再进需重新满槽——点燃时机=决策)/ 硬撞坠落;滞空、吃弹、擦墙均不断。** 最后 20 点必须靠战斗收入。表现:色调+音乐变奏+近肩相机+满槽 READY 脉冲。

### 4.6 攻击系统:斩击,一刀多结算

主动词=**出剑**(格挡语义废):你攻击子弹,把它劈回去。
**判定规格(铁律):** 可斩目标=CollectionService Tag 注册的脚本对象;按键帧做**车体局部盒查询**(前 12/侧 ±7/上下 ±5/后 -2)——**禁物理 Touched**。剑光=纯 VFX 零耦合;子弹=发射瞬间固定弹道快照,**永不追踪**(变速变线本身即闪避);弹反=改写目标为射手。窗口=盒深÷车速(越快越紧,技巧随速自动升级);单击结算盒内**全部**目标(多杀=关卡师排布的高光);服务端距离合理性校验(M12)。
**反馈铁律:反馈永不冻结画面与输入**——gameplay 禁子弹时间/hitstop/任何时间膨胀(135 速 0.1s 冻结=偷 13 studs 读路窗口);弹反爽感=音效尖峰+闪光色差+子弹肉眼可见飞回杀射手;慢镜头只进结算页与回放。
**敌人物性:** 远程敌本体=无碰撞脚本对象,骑穿零惩罚,威胁全部来自子弹,可摆路面正中(迎面弹=最可读的"物体飞向我"弹道);**敌人身体永不做碰撞障碍**;**只对玩家前方目标开火、骑过即沉默**(背后追击弹否决:闭合速度仅 15 studs/s 追不上快玩家=只罚新手,且不可读;ADR-38 定稿)。

**交互矩阵:**

| 目标 | 窗口 | 结算 | 能量 |

| 远程敌·子弹 | 时间(到达瞬间,预告线) | 劈回子弹→击杀射手 | +25(破 Cap,击杀帧入账) |
| 远程敌·本体 | 空间(骑到近前) | 直接斩杀;骑穿=穿透零惩罚 | +30(破 Cap) |

| 能量闸门(可破坏门) | 空间 | 劈开通行;**未斩=实体墙,走碰撞三档**(主线摆放必可绕行 ≤半幅,§4.7/§7) | +15 |
| 捷径挡板 | 空间 | 劈开进捷径 | 0(捷径即奖励) |

### 4.7 墙与门家族(ADR-42 定稿:三型,物性各自干净,无中间态)
- **纯障碍(承重墙/挡板体):** 不可砍、不打 Tag;绕行是唯一解,撞上走碰撞三档。
- **可破坏门(`Destructible`):** 斩开=通行 +15(红区收入);**未斩=实体墙,走碰撞三档**;R 重开复原。主路线上必可绕行(≤半幅);全幅封堵仅限**教学闸门**(斩击键教学锚,教程 100% 学会)与**捷径挡板**(捷径的技巧税,劈开进捷径=0 收入,捷径即奖励)。
- **节点电源门(V2·节点联动墙):** 斩路边电源节点→电缆光束→前方门开 N 秒(倒计时灯条)→闭合;节点与门**发光电缆连接+同色编码**=视觉铁律;`Combat_GateLinkWindowSec=2.5` 预埋。
"穿透减速式收费站闸门"已废(ADR-42)。

### 4.8 转向辅助开关(底层安全网)
自动修线+提前减速;保安全但必然更慢——"关辅助"=底→中的仪式。辅助不代骑贴墙段。

### 4.9 贴墙段(授权段式表面吸附)
**形态(GR2/MK8 语法):** 可贴墙面=带 `WallRideSurface` Tag 的授权墙段(直墙+90° 圆角过渡弯先行——圆角=内接折线小面片链、单关节 ≤8° 门禁不放宽、控制器折线行走跨段,ADR-43;任意曲墙随 TrackBuilder P3;倒挂/天花板/管道=V3 上轨演出段)。**磁吸模型:** 墙态内 down=−墙法线,速度不变;A/D=墙上高度带;进入=坡唇/触发区(地面或空中皆可,0.2s 位置姿态混合,有速度阈值);退出=段尾弹道交接 / 驶下下缘回地 / 驶出上缘弹道——**墙上高度=落差跳距**(高台弹射,§7)。相机 roll≤90° 平滑滚转,FOV 不变。
**经济与判定:** 墙=**经济中性**,墙态与任何墙体接触零入账("贴墙就是平地跑");未入墙态的接触=纯惩罚照旧。进出与墙上滞空不断心流;坠落按 fell 断;墙段不烘焙锚点(坠落回段前);硬核=回分段门。
**存在理由(soft lint):** 墙段必须在路线上有意义——捷径/更顺走线/高台弹射/躲弹/演出,否则无人上墙;墙段内不摆判定点与射手。

## 5. 手感与数值基准(Config 实时可调)

## 6. 相机与视角

## 7. 关卡设计语法(TrackBuilder 的 lint 清单)

- 高速段宽走廊;截面微谷形;地标+灯带导航。
- 捷径全部是有意设计的风险-回报缺口;好捷径入口配挡板(技巧税)。
- **斩击/弹反判定点只放直线段,永不与转向决策重叠**(同时决策数峰值锁 2;移动端右拇指永不双键)。
- **主路线永不强制斩击:** 强制斩击仅限教学闸门与捷径挡板;路面敌与能量闸门必须可绕行或可穿透。
- **物性 lint:** 一切可骑面必须带 Rideable Tag(TrackBuilder 自动);禁止"骑不上去又不算墙"的模糊几何。
- 水晶贴主行车线(**磁吸=保底的实现**,顺路即得);**能量核摆偏线与危险位**;新手赛道加密水晶、硬核赛道多核——供给密度=难度旋钮。
- **供给两桶 lint:** 蓝区(水晶×磁吸)/红区(战斗:核/敌/闸门)分列;**红区=0 报警**(心流不可达);参考密度:晶 12~18/核 3~6/敌 3~5/闸 0~2(M9.5 标定);蓝区密度=底层冲刺口粮(埋点"能量归零时长占比"=阀门信号)。
- 弯道按"全速+好走线可过"设计,刹车永远只是容错余量(弯道半径 lint 按心流速 145.8 校验,阈值 ≈167 studs)。
- **跳跃段语法:** 同高落点近似跳距 ≈2·v²·tanθ/g(g=260):20° 坡≈巡航 28/冲刺 51/心流 60 studs,30° 坡≈44/81/94。**平断口(θ=0)同高对岸永远跳不过**;断桥三形态=①平断口+对岸下沉(≈v·√(2h/g))②断口带 5~15° 起跳唇③坡顶断口。**缺口设计法:巡航过不去/冲刺刚好/心流余量**(空中松开冲刺=距离骤减,抉择即难度);精确数字跑 `tools/jumptable.lua`;跳跃段上空无锚点。
- **贴墙段语法:** 用法=捷径线/高台弹射/躲弹与演出;soft lint=墙段必须在路线上有意义;墙段内不摆判定点/射手;教程赛道含一段入门墙(贴墙 beat=移动教学)。
- 敌人密度 3~5/条,固定摆放确定性刷新;R 重开全部重置(验收断言)。
- **分段门 3~5 个/条**(60~90s 赛道):可见霓虹拱门,兼服务端校验门与岔路导航;差值显示规格见 §3。
- **节拍配比(初版模板,ADR-38 会话拍板,M9 试玩迭代):** 90s ≈ 发车直线 8s(读路+水晶列)→ 技巧弯组 14s(全速 S 弯)→ 战斗直线 10s(敌×2+核)→ 爽段 12s(长下坡+坡顶跳)→ 分叉段 10s(主线宽弯/贴墙捷径 ~3s)→ 技巧弯组 14s(变奏弯+闸门)→ 高潮连段 12s(跳台→墙段→落点双核)→ 冲线直线 10s;占比≈爽 35%/技巧 30%/战斗 15%/缓冲 20%;供给对齐两桶参考密度(敌 3~4/核 3~4/晶 ~14/墙段 2≈5s/闸门 1)。

## 8. 大厅与进入流程(完整大厅=M11 后置;V1 demo=直进赛道最小入口)
